# Optimización de sitios Hugo para SEO

El SEO (Search Engine Optimization) es esencial para que tu blog sea visible y fácil de encontrar en motores de búsqueda como Google. Hugo, al ser un generador de sitios estáticos, permite implementar prácticas SEO de manera sencilla y eficiente.

En este artículo veremos **técnicas prácticas para mejorar el SEO de un sitio Hugo**, desde la configuración de títulos y meta tags, hasta la estructura de contenido y la generación de sitemap.

---

## Títulos y meta description

Los títulos y las descripciones son la primera impresión que los motores de búsqueda muestran en los resultados. Hugo permite definirlos en el front matter de cada página:

```yaml
title: "Optimización de sitios Hugo para SEO"
description: "Aprende a mejorar la visibilidad de tu blog Hugo con técnicas de SEO prácticas."
```

Recomendaciones:
- Mantén títulos claros y descriptivos, incluyendo palabras clave relevantes.
- Cada página debe tener una meta description única y atractiva.
- La longitud recomendada: títulos entre 50-60 caracteres, meta description entre 150-160 caracteres.
- Evita duplicar títulos o descripciones entre páginas.
- Esto ayuda a los motores de búsqueda a entender de qué trata cada página y mejora la tasa de clics desde los resultados.

---

## URLs limpias
Las URLs amigables son esenciales para SEO y experiencia de usuario. Hugo permite personalizar las URLs mediante el front matter o la configuración global.

Buenas prácticas:
- Evita caracteres especiales o acentos.
- Usa guiones - en lugar de espacios.
- Mantén URLs cortas y descriptivas, por ejemplo:


```yaml
slug: "seo-hugo"
```

Esto generará una URL como:

```arduino
https://tusitio.com/seo-hugo/
```

En lugar de:

```perl
https://tusitio.com/optimización%20sitios%20hugo/
```

---

## Estructura de contenido

La jerarquía de títulos y subtítulos es fundamental para SEO y accesibilidad:
- Usa un H1 por página (normalmente el título del artículo).
- Utiliza H2 para secciones principales y H3 para subsecciones.
- Las listas, tablas y subtítulos mejoran la legibilidad y facilitan que los motores de búsqueda indexen correctamente el contenido.

Ejemplo:

```markdown
# Optimización de sitios Hugo para SEO
## Títulos y meta description
### Cómo escribir meta descriptions efectivas
```

---

## Sitemap y robots.txt
Hugo genera automáticamente un sitemap.xml, que ayuda a los motores de búsqueda a indexar todas las páginas de tu sitio.

- Asegúrate de habilitarlo en la configuración:

```toml
[sitemap]
  changefreq = "monthly"
  priority = 0.5
  filename = "sitemap.xml"
```

- Configura robots.txt para controlar qué páginas deben ser indexadas:

```txt
User-agent: *
Disallow: /admin/
Allow: /
Sitemap: https://tusitio.com/sitemap.xml
```

>Esto optimiza la indexación y evita que motores de búsqueda rastreen contenido innecesario.

## Otros consejos SEO para Hugo

- Etiquetas y categorías: ayudan a organizar el contenido y mejorar la navegación.
- Imágenes optimizadas: usa alt descriptivos y reduce el tamaño para mejorar velocidad de carga.
- Velocidad del sitio: Hugo genera sitios estáticos ultrarrápidos, lo que mejora tu ranking.
- Enlaces internos y externos: enlaza páginas relacionadas y fuentes de autoridad.

## Conclusión

Seguir estas prácticas en tu blog Hugo mejora notablemente la visibilidad en motores de búsqueda, la experiencia del usuario y la accesibilidad.

Hugo, con su estructura limpia y generación de sitios estáticos, facilita implementar SEO técnico sin depender de plugins complejos, asegurando que tu contenido sea indexado correctamente y de manera eficiente.

> Optimizar tu sitio Hugo para SEO no solo ayuda a los motores de búsqueda a entender tu contenido, sino que también mejora la experiencia de tus lectores.
